# Security Policy

## Supported Versions

Only the latest published release and the `main` branch are supported with security updates.

## Reporting a Vulnerability

Please do **not** open a public issue for suspected vulnerabilities.

Report privately by email to: **bart@waardenburg.dev**

Include:
- Affected repository and version/commit
- Reproduction steps or proof of concept
- Impact assessment

You will receive an acknowledgement as soon as possible, followed by remediation and coordinated disclosure when applicable.
